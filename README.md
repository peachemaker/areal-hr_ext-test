## Инструментарий разработки

1. Операционная система: Windows 11 с установленным WSL2 и Docker Desktop
2. IDE: Visual Studio Code
3. PostgreSQL: используется docker контейнер

## Основные git команды
* git clone <url>          # Клонировать репозиторий
* git status               # Статус файлов
* git add .                # Добавить все изменения
* git add file.ts          # Добавить конкретный файл
* git commit -m "message"  # Коммит с сообщением
* git push                 # Отправить на сервер
* git pull                 # Получить изменения
* git branch               # Список веток
* git checkout -b feature  # Создать и переключиться
* git checkout main        # Переключиться на main
* git merge feature        # Слить ветку
* git branch -d feature    # Удалить ветку
